# CAD 기반 건축 매스 생성 및 일조 분석 시스템 (PRD)

## 1. 프로젝트 개요

본 프로젝트는 **CAD 도면을 입력받아 자동으로 건축 매스를 생성하고, 실제 부지 위에 배치하여 일조 및 법규 조건을 검토할 수 있는 웹 서비스**를 개발하는 것을 목표로 한다.

사용자는 CAD 도면(DXF/DWG)을 업로드하면 시스템이 도면을 분석하여 건물 매스를 생성하고, 이를 지도 위에 시각화한다.
또한 태양 위치를 기반으로 일조 시뮬레이션을 수행하여 건축 계획 단계에서의 검토를 지원한다.

---

## 2. 문제 정의

건축 설계 초기 단계에서 다음과 같은 문제가 발생한다.

* 도면 기반 매스 검토 과정이 수작업 중심
* 부지 주변 건물과의 관계 분석 어려움
* 일조권 분석에 시간이 많이 소요됨
* 법규 조건 검토가 분산된 정보로 진행됨

따라서 **도면 기반 자동 매스 생성 및 분석 시스템**이 필요하다.

---

## 3. 목표

### 핵심 목표

* CAD 도면을 자동으로 분석하여 **3D 매스 생성**
* 생성된 매스를 **지도 위에 배치**
* **일조 시뮬레이션 제공**
* 부지 기반 **법률 조건 검토 기능 제공**

### 성공 기준

* CAD 도면 입력 → 매스 생성 성공
* 지도 위 실제 좌표에 건물 배치
* 시간 기반 태양/그림자 시뮬레이션 가능
* 간단한 법규 검토 기능 제공

---

## 4. 주요 기능

### 4.1 CAD 도면 업로드

입력 지원

* DXF
* DWG (변환 후 사용)

기능

* CAD 파일 파싱
* 건물 외곽선 추출
* 레이어 기반 벽 인식

---

### 4.2 건물 매스 생성

CAD 도면을 분석하여

* 건물 footprint 추출
* 층고 설정
* 3D 매스 생성

출력

* glTF / GLB 모델

---

### 4.3 지도 기반 시각화

기능

* 지도 로드
* 부지 선택
* 건물 매스 배치
* 주변 건물 표시

---

### 4.4 일조 시뮬레이션

기능

* 태양 위치 계산
* 시간 기반 그림자 생성
* 건물 차폐 여부 확인

입력

* 날짜
* 시간

출력

* shadow visualization
* 일조 분석 결과

---

### 4.5 법규 조건 검토 (기초)

예시

* 건폐율
* 용적률
* 대지면적

입력

* 지적도
* 건물 footprint

출력

* 법규 충족 여부

---

## 5. 사용자 시나리오

### 시나리오 1

건축 설계자가 CAD 도면을 업로드한다.

시스템은 도면을 분석하여 건물 매스를 생성한다.

---

### 시나리오 2

사용자는 부지를 선택한다.

건물 매스가 실제 좌표에 배치된다.

---

### 시나리오 3

사용자는 날짜와 시간을 선택한다.

시스템은 태양 위치를 계산하여 그림자를 생성한다.
