# System Architecture

## 전체 시스템 구조

```
                User
                 │
                 ▼
          Web Frontend
             (React)
                 │
                 ▼
           Cesium Viewer
                 │
                 ▼
        3D Model Loader (glTF)
                 │
                 ▼
              Backend
            (FastAPI)
                 │
        ┌────────┴────────┐
        ▼                 ▼
   CAD Parser        Data Services
   (ezdxf)          (GIS / 지적도)
        │
        ▼
  Geometry Generator
        │
        ▼
     Mesh Builder
        │
        ▼
    glTF Exporter
        │
        ▼
      Storage
```

---

## Frontend

### 주요 역할

* 지도 렌더링
* 3D 모델 시각화
* 사용자 인터랙션

### 구성

```
React
│
├─ Cesium Viewer
├─ Model Loader
├─ Shadow Controller
└─ UI Controls
```

---

## Backend

### 주요 역할

* CAD 도면 처리
* 매스 생성
* 모델 변환

### 구성

```
FastAPI
│
├─ CAD Parser
├─ Geometry Processor
├─ Mesh Generator
├─ glTF Exporter
└─ Simulation Engine
```
