===============================================================================

  - Mapped[] type hints instead of Column() definitions
  - DeclarativeBase instead of declarative_base()
  - mapped_column() for column definitions
  - Type-safe ORM queries

✓ UUID Primary Keys
  - All tables use UUID (not auto-increment integers)
  - Better for distributed systems and privacy

✓ UTC Timestamps
  - All datetime fields use DateTime(timezone=True)
  - Server defaults use func.now() for UTC
  - Automatic onupdate for updated_at columns

✓ Proper Relationships
  - One-to-many relationships with back_populates
  - Cascade delete to maintain referential integrity
  - Lazy loading configured appropriately

✓ Indexes

  - Performance-optimized for common queries

✓ JSON Columns
